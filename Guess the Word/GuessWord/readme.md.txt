Guess The Number :
It's a Guess the number game with loops, where obviously you have to guess a number. It's made with the loops concept along with if/else statement concept and also with a limit of 3 guesses only to make the game more good. The loop will run till you guessed the actual word or you are out of guesses.

Concepts used in this project :
Boolean Value.
If/else statement.
While Loops.
Language which is used :
This project is built in Python and you can download it from here.

How to play it?
Copy the given code and paste it in your code editor, for instance: VS Code, Sublime and PyCharm etc, and then run/play it and u can play the game in your terminal and Guess the actual number.
